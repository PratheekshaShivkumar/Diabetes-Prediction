# Diabetes-Prediction
# 🩺 Diabetes Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a person is likely to have diabetes using Machine Learning techniques. The model is trained on a public dataset to classify patients based on various medical attributes.

## 🎯 Objective
To develop a machine learning model that can accurately predict diabetes risk based on patient health data.

## 📊 Dataset
- Source: Kaggle
- Features include:
  - Pregnancies
  - Glucose
  - Blood Pressure
  - Skin Thickness
  - Insulin
  - BMI
  - Diabetes Pedigree Function
  - Age
- Target:
  - Outcome (Diabetic / Non-Diabetic)

## 🛠 Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🔄 Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Diabetes Prediction

## 🤖 Machine Learning Model
    - Logistic Regression
## 📈 Results
Accuracy:75%

The trained model predicts whether a patient is diabetic based on the given medical parameters.
